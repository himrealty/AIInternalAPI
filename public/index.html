API Endpoints

POST/prompt/:provider
Body: {"message": "..."} — Send prompt to any provider. Claude is routed to promptui automatically.
GET/prompt/:provider?message=...
Same as above, GET version
POST/run
Body: {"provider":"...","action":"...","message":"...","chatIndex":0,"imageSize":"16:9"} — Full control
GET/status
Server health + extension connection state
GET/providers
Returns provider → supported actions map (mirrors extension config)

curl Examples

POST prompt to Claude (auto-routes to promptui)
curl -X POST /prompt/claude \
  -H "x-api-key: YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"message": "What is 2+2?"}'
Full control via /run — get chats from DeepSeek
curl -X POST /run \
  -H "Content-Type: application/json" \
  -d '{"provider":"deepseek","action":"getchats"}'
Qwen image generation with aspect ratio
curl -X POST /run \
  -H "Content-Type: application/json" \
  -d '{"provider":"qwen","action":"qwenimage","message":"a sunset over mountains","imageSize":"16:9"}'
